# SVN分支创建与checkout小工具

svntools is a tool for the svn. It can help you quickly create a branch,and checkout

## Usage

* 将文件拷贝至任意目录，解压后获得`svntools.sh`
* 打开终端
* 运行脚本`bash svntools.sh`  或直接 `./svntools.sh`
* 按引导完成分支的创建和checkout
